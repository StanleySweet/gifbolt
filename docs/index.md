# GifBolt Documentation

Welcome to the GifBolt documentation.

- Project: GPU-accelerated GIF renderer for WPF and Avalonia.
- Native C++17 core, .NET 6 wrappers.

API references:

- C++ API: available after CI build under “API Reference → C++ API”.
- C# API: available after CI build under “API Reference → C# API”.

See also the version page: [VERSION_API.md](VERSION_API.md).
