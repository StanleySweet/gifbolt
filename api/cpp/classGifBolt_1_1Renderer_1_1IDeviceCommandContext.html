Detailed Description

Abstract interface for rendering device commands and GPU resource management.

Inspired by 0 A.D.'s architecture, provides backend-agnostic rendering abstraction. Enables pluggable rendering backends (DirectX 11, dummy, etc.) for maximum flexibility.
