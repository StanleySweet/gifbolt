Gets BGRA32 pixel data with premultiplied alpha for the specified frame, scaled to target dimensions.

Parameters
decoderThe decoder handle.
indexThe zero-based frame index.
targetWidthThe desired output width in pixels.
targetHeightThe desired output height in pixels.
[out]outWidthPointer to receive the actual output width.
[out]outHeightPointer to receive the actual output height.
[out]byteCountPointer to receive the size of pixel data in bytes.
filterTypeThe scaling filter to use (0=Nearest, 1=Bilinear, 2=Bicubic, 3=Lanczos).
Returns
Pointer to BGRA32 premultiplied scaled pixel data, or NULL on error. The pointer is valid until the next decoder operation.
